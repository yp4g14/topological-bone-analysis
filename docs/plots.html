Analysis plots¶

src.plots.analysis_plots(df, filenames_map, plot_path, feature_cols)[source]¶

Creates box plots to compare groups of statistics, using filenames_map.

Parameters

  • df (pandas DataFrame) – df containing filename, patch_number, quadrant, category columns as well as any statistics columns you want to plot

  • filenames_map (dict) – dictionary with keys containing filenames (strings) and values containing group id

  • plot_path (string) – location to save plots as .svg

  • feature_cols (list) – list of strings, statistic names to plot
