Persistent Homology SEDT¶

src.persistent_homology_SEDT.peristent_homology_sublevel_cubic(image, filename, save_path, plot_persistence_diagrams=False)[source]¶

Calculate sublevel set cubical homology persistence for an image Image should already have undergone SEDT for SEDT transform filtration Takes the idiagram file and pulls out the birth,death persistence intervals for two dimensions (0,1). Saves the persistence intervals as csv. Optionally plots the persistence diagrams and saves as svg.

Parameters

  • image (numpy array) – image values along which to take the filtration

  • filename (string) – filename for image as string

  • save_path (string) – location to save necessary files as string

  • plot_persistence_diagrams (bool, optional) – If True saves plots of the persistence diagram. Defaults to False.
