src.topological_bone_analysis.permutation_hypothesis_test.multiple_comparison_correction(p_vals_path, stats_list, logger, category_pair, alpha=0.05)[source]¶

This function corrects for the multiple comparison problem between two categories (specified by category_pair) by using a Benjamini-Hochberg p-value adjustment at an overall significance (specified by alpha).

Parameters

  • p_vals_path (string) – location of p values csv as string

  • stats_list ([type]) – lsit of statistics to adjust over

  • logger (logger.Logging object) – logging

  • category_pair (tuple) – tuple of category names for which to make the adjustments e.g. (‘group_a’,’group_b’) takes all tests between these two groups and adjusts for the stats in stat_list

  • alpha (float, optional) – Level for p-value adjustment. Defaults to 0.05.

Returns

dataframe with original and adjusted p vals,

sorted by increasing adjusted p value.

Return type

pandas DataFrame
