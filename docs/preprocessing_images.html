src.preprocessing_images.extract_patches(image, patch_shape, pad_val=0, stride=None)[source]¶

Takes a 2D image array and cuts it into non-overlapping square patches of patch_shape. To do this it first pads image to be exact multiples of the patch shape in each direction. The image is padded with the constant pad_val, which defaults to 0

Parameters

  • image (numpy array) – numpy array of image

  • patch_shape (int) – length (width) of desired square patches.

  • pad_val (int, optional) – value in [0,1] that pads the binary image. efaults to 0.

  • stride (int, optional) – stride across image before taking next patch. Defaults to patch_shape so patches don’t overlap

Returns

(array) padded image patches : (array) array of patches, patches[i] will return i^th patch. patch_coords : list of all patch coordinates (top left per patch)

Return type

image
